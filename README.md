# GHG Accounting Model for Regenerative Agriculture Supply Chains

This project demonstrates a simplified Python-based GHG accounting workflow for agricultural operations and farm-level supply chain comparison.

## Features
- Scope 1 stationary emissions
- Scope 1 fleet emissions
- Farm-level emissions comparison
- Regenerative vs conventional intensity analysis

## Run
```bash
pip install -r requirements.txt
python src/ghg_accounting.py
